Statistics about Dragon Age on AO3

The "Archive of Our Own" (AO3), a nonprofit open source repository for fanworks, has become one of the most popular archives for transformative works since its founding in 2008. On this page I have collected several statistics about the fanfictions of the Dragon Age Universe [1].This was achieved by going through the tag "Dragon Age - All Media Types" on AO3. This means that any work not tagged with this was not considered.

Publishing dates

Every work on AO3 has a date which describes when the fanfiction was last updated. For single-chapter stories this is usually the same as the post date. For multi-chapter fanfictions the date shows the publishing date of the last chapter. Characters

AO3 allows the authors to tag the characters which appear in their story. They can use what ever tag they like, but the most popular characters usually get suggested once the author starts typing to prevent users from creating several tags for one and the same character. This does not always work, however. A popular example are the tags for Solas from Dragon Age: Inquisition. For him the following tags exist (sorted by popularity with the most popular one coming first): "Solas", "Solas (Dragon Age)", "Fen'Harel | Solas" plus an additional set of tags specifying how Solas behaves in the certain work (for example "dark!Solas").
There are authors who will just use one of those tags, while other authors might use more than one tag for the character. Due to that the amount a tag is used does not necessarily equal the amount of fanfictions written about that character. But it does give an estimate for popularity of a character.

Types of main character in Dragon Age: Inquisition

In Dragon Age: Inquisition the player can choose from four different races for the main character ("The Inquisitor"). These are human (last name Trevelyan), elf (last name Lavellan), dwarf (last name Cadash) and qunari (last name Adaar)[2]. In the following graph I chose the most popular set of tags to estimate how many fanfictions there are with the specific races. However, since there are some fanfictions who do use more than one Inquisitor tag these are only estimates. Also, there were over 50 different tags for Inquisitors, ranging from only "Inquisitor" up to combinations for gender, race and class. In addition many authors did use the specific first name of the inquisitor, which can be chosen freely.
The tags which were chosen to represent were:

Female [last name], Male [last name], [last name]
Female Inquisitor (Dragon Age), Male Inquisitor (Dragon Age), Inquisitor (Dragon Age)

Note: Nonbinary/Genderqueer/Agender tags were not included in this graph. While the "Nonbinary Inquisitor" tags exist, they are not used in a capacity that would make them particularly visible in the graphic. For completion purposes their numbers will be added here:
Nonbinary Inquisitor (Dragon Age): 21, Nonbinary Lavellan: 12, Nonbinary Adaar: 6, Nonbinary Trevelyan: 2, Nonbinary Cadash: 0

Most popular ship tags

In addition to characters AO3 also lets the author tag relationships (usually just called "ships") which appear in their story. These are usually in the format "[first character] / [second character]". While most tagged relationships are of romantic or sexual nature it is also possible to tag friendships, which is usually done in the format "[first character] & [second character]". Length and chapters

AO3 gives the authors the ability to upload their works in chapters. While many users use this function there are a number of authors who upload their whole work in one chapter. While the shorted uploaded work has 0 words, the longest one has currently 996176 words (19 Sep 2019). A lot of works with 0 words are not written ones but instead contain fanart which is not counted in the word count. In addition to this works can also be marked as "completed", signaling the user that the story is completely finished.
Looking at the distribution of work length one sees that a large percentage of works has a word count of 90000 and under (~98.1%). Due to that I will only work with these in the following. For the graph I sorted the works under 90000 words into 50 equally sized bins. The dots do show how many chapters the works in these buckets have in average. By clicking on the button the dots change to display how many of these works are marked as "completed". Hovering over the dots at any time shows a tooltip with the statistics for each bin.
